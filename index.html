    Veri Kurtarma (Eski Hesaba Bağlan)

    Netlify'dan GitHub'a geçtiğiniz için sistem size boş bir ID verdi. Eski sorularınız Firebase'de duruyor.

    1. Firebase Console'a girin.
    2. Firestore Database > Data sekmesine gidin.
    3. artifacts > rana-lgs-desktop-html > users yolunu izleyin.
    4. Orada yazan uzun ID kodunu (örn: xYzA123...) kopyalayıp aşağıya yapıştırın.
